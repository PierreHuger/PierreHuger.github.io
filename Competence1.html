Puissance 4

Le projet du Puissance 4 se divise en 5 étapes. Dans un premier temps, j'ai réalisé une maquette. Grace a ma maquette, j'ai pu ensuite me concentrer uniquement sur le développement et je n'ai pas eu à imaginer le rendu visuel de mon Puissance 4.

J'ai ensuite réalisé un cahier de test afin d'imaginer tous les cas d'erreur ou les cas impossibles. Ce cahier, je le réutiliserai plus tard dans le projet, dans la phase de test.

Puis je suis passé au développement de mon Puissance 4. Le Puissance 4 a été développé en C et j'ai utilisé Visual Studio Code comme support de développement.

Pour finir ce projet, j'ai réalisé les tests a partir de mon cahier de tests que j'avais réalisé précédemment dans le projet.
